### Manual Installation

If you prefer to install manually, copy the component code and install these dependencies:

```bash
npm install motion/react @use-gesture/react
```

## Features

- **Interactive Stacking**: Images stack behind each other with proper z-index layering
- **Click & Drag Navigation**: Both click and drag gestures for navigation
- **Mobile Optimized**: Touch-friendly with responsive drag thresholds
- **Spring Animations**: Smooth spring physics for natural feel
- **Rounded Corners**: Smart rounding - only visible edges get rounded corners
- **Content Overlay**: Text and buttons with gradient mask for readability
- **Responsive Design**: Adapts to different screen sizes

## Usage

```tsx
import StackedTabs from "@/components/ui/stacked-tabs";

export default function Example() {
  return <StackedTabs />;
}
```

## API Reference

### Props

| Prop           | Type           | Default         | Description                        |
| -------------- | -------------- | --------------- | ---------------------------------- |
| `images`       | `ImageData[]`  | Built-in images | Array of image objects             |
| `buttonWidth`  | `number`       | `96`            | Width of inactive panels in pixels |
| `springConfig` | `SpringConfig` | See code        | Custom spring animation settings   |

### ImageData Interface

```tsx
interface ImageData {
  src: string; // Image URL
  title: string; // Main title text
  buttons: string[]; // Array of button labels
}
```

## Customization

### Animation Settings

You can customize the spring animation by modifying the transition properties:

```tsx
transition={{
  type: "spring",
  stiffness: 400,    // Higher = faster response
  damping: 35,       // Higher = less bounce
  mass: 0.8,         // Lower = lighter feel
  duration: 0.3,     // Animation duration
}}
```

### Mobile Configuration

The component automatically adjusts for mobile devices:

```tsx
const isMobile = window.innerWidth < 768;
const mobileThreshold = isMobile ? 50 : 100; // Drag sensitivity
```

# Introduction

Welcome to **Hex UI**, a modern React component library inspired by hexagons because they are cool and efficient. Built with Next.js, TypeScript, and Framer Motion for smooth animations.

## What is Hex UI?

Hex UI provides a collection of beautifully designed, accessible, and customizable React components with smooth animations. All components are built with modern web standards and can be easily integrated into your Next.js projects.

## Key Features

- **Beautiful Design**: Clean, modern components with attention to detail
- **Smooth Animations**: Built with Framer Motion for fluid interactions
- **Fully Customizable**: Easy to customize with Tailwind CSS
- **Accessible**: Built with accessibility in mind
- **Responsive**: Works seamlessly across all devices
- **TypeScript Ready**: Full TypeScript support out of the box

## Installation Methods

Hex UI components can be installed using the shadcn CLI for seamless integration:

```bash
npx shadcn@latest add https://hex-ui.vercel.app/r/component-name.json
```

## Component Categories

### Text Animations

Beautiful text animations including fade, slide, blur, and transform effects.

### Interactive Elements

Buttons, cards, and other interactive components with hover effects.

### Layout Components

Responsive layout components for building modern interfaces.
